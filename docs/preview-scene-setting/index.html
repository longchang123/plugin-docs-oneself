场景预览助手

Preview-Scene-Setting

插件说明

游戏开发过程中,有时经常需要在场景之间进行切换来进行一些开发工作,使用该插件可以快速进行初始预览场景的设置.

举个栗子

比如我写了一个TestScene场景,这个场景我会加入一些我自己的测试代码,然而在启动运行项目的时候,可能我需要设置初始预览场景为loginScene,为此我还需要打开菜单项目=>项目设置=>项目预览=>初始预览场景

用了这个插件之后,只需要简单切换一下即可

其他说明

该插件推荐全局安装
